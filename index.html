Stuff I've worked on

  • Context-Sensitive Incremental Points-To Analysis (2019)

    With a partner, I improved the precision of a state-of-the-art incremental algorithm for points-to analysis.

  • CUDA DSL + Compiler (2018)

    With a partner, I designed a language to simplify writing massively parallel code, and implemented a compiler targeting CUDA.

  • OCaml-to-Blockchain compiler (2018)

    I wrote a compiler from OCaml bytecode to EVM bytecode. It was lots of fun.

  • Equation Grapher (2014)

    I wrote an equation grapher in Javascript on the educational site Khan Academy. The user can type in custom equations and see their graphs. The fun part of this project was the fact that Khan Academy disallows eval.

  • Tartan Tag (October 2015)

    At HackCMU, I worked in a team of 4 to make this Android app. It's an icebreaker-ish sort of game. We used Google's Nearby Connections API to create a local game server for two teams. Every player prints two UPC-A barcodes, attaching one to their back and one to their front. Then, tag!—and not just any tag, but tag with cameras—and not just any cameras, but the cameras on your phones.
